# 🐛 Seu Barriga do Zap

> **"Won't pay? The bot collects."**

A collection of creative and fun solutions to remind that deadbeat friend on WhatsApp — without manually sending messages every hour.

Inspired by **Seu Barriga** (*Mr. Belly*), the iconic landlord from the Brazilian TV show *Chaves*, who never gave up collecting rent. 🏠

## 🎯 What is this?

Each **solution** here is a different way to automate WhatsApp messages. It could be a browser console script, an n8n workflow, a Python bot, an AI agent... The rule is: **creative, functional, and fun**.

Use it to collect debts, confirm RSVPs, run polls — or any situation where you need a reply and the person just *vanishes*.
